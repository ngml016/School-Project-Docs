# 🅲 学生端 UI 主题 · 极简黑白

## 风格描述
干净、现代、不花哨。无渐变、无彩色装饰，纯黑白灰层次感。加载速度最快。

## 色彩系统

### 主色调
- 主色：`#1A1A2E`（深蓝黑） 主色浅：`#F0F0F5` 主色深：`#000000`

### 辅色
- 辅色：`#4A4A6A` 辅色浅：`#F5F5F8`

### 强调色（唯一彩色）
- 危险：`#E53E3E`

### 背景色
- 页面：`#FAFAFA` 卡片：`#FFFFFF` 卡片交替：`#F5F5F5`

### 文字色
- 标题：`#000000` 正文：`#333333` 次要：`#666666` 占位：`#999999` 反白：`#FFFFFF`

### 边框
- 边框：`#DDDDDD` 浅边框：`#EEEEEE` 分割线：`#EEEEEE`

## 排版
- 英文字体：`-apple-system, BlinkMacSystemFont, 'Inter'`
- 中文字体：`'PingFang SC', sans-serif`
- 字号：11/13/15/17/20/24/28px
- 字重：400/500/600/700

## 间距
- 基础 4px，与主题 A 保持一致

## 圆角（最小化）
- 小：4px 中：8px 大：12px 超大：20px 全圆：999px

## 阴影（最轻）
- 轻：0 1px 2px rgba(0,0,0,0.04)
- 中：0 2px 8px rgba(0,0,0,0.06)
- 重：0 4px 16px rgba(0,0,0,0.08)
- Toast：0 4px 20px rgba(0,0,0,0.15)

## 组件样式要点
- 主按钮：纯黑底、白字、小圆角4px、无渐变、按压 opacity0.7，高度42px
- 输入框：白底、1px灰边框、最小圆角、聚焦变纯黑、**无光晕**
- 卡片：白底、8px圆角、1px细边框、**无阴影**
- 高亮卡片：左侧3px黑色竖条
- Tab：**直角分段控制器**（非胶囊），纯黑激活态
- Toast：右上角、纯黑底、最小圆角4px、0.35s滑入
- 通知列表：未读加粗、已读 opacity0.6（不用背景色）
- 操作记录：纯分割线列表，无卡片阴影
- 课表：时间用等宽字体 `'SF Mono', monospace`
- Badge：灰底黑字，极小圆角
- 图标：使用 emoji
- 移动端：max-width 480px
