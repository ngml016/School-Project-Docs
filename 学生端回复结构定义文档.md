# 🎒 学生端大模型回复结构定义

> **核心原则：SSE 流式传输，事件类型驱动，前端按类型渲染。**
>
> Agent 的回复分为三种类型：纯文本消息、卡片结构数据、工具调用状态。前端根据 SSE 事件类型分别处理，不依赖启发式检测。

---

## 📡 SSE 事件协议

后端 `POST /api/chat` 以 SSE (Server-Sent Events) 流式返回，每个事件一行：

```
event: {event_type}
data: {json_data}
```

### 事件类型一览

| 事件类型 | 含义 | 前端处理 |
|:---------|:-----|:---------|
| `message` | Agent 回复的文本片段 | 逐段追加到当前气泡 |
| `card` | 结构化数据卡片 | 渲染为卡片组件 |
| `tool_call` | Agent 正在调用工具 | 显示"思考中..."状态 |
| `tool_result` | 工具调用完成 | 更新为结果（可选） |
| `error` | 处理出错 | 显示错误提示 |
| `done` | 回复结束 | 停止流式接收，启用输入 |

---

## 1️⃣ `event: message` — 纯文本回复

### 格式

```
event: message
data: {"content": "签到成功！高等数学 ✅"}
```

### 完整示例

```
event: message
data: {"content": "好的，我来帮你查看今天的课表～"}

event: message
data: {"content": "\n\n📅 周三 6月26日"}

event: message
data: {"content": "\n08:30-10:00 高等数学 @ 教学楼301"}

event: message
data: {"content": "\n10:20-11:50 大学英语 @ 教学楼205"}

event: done
data: {}
```

### 前端处理

```typescript
// 逐段追加到当前 assistant 气泡
let currentBubble = "";
const reader = resp.body!.getReader();
const decoder = new TextDecoder();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;

  const chunk = decoder.decode(value);
  for (const line of chunk.split("\n")) {
    if (line.startsWith("data: ")) {
      const data = JSON.parse(line.slice(6));
      if (data.content) {
        currentBubble += data.content;
        // 触发 React 重新渲染
        setMessages(prev => {
          const last = prev[prev.length - 1];
          if (last?.role === "assistant") {
            return [...prev.slice(0, -1), { ...last, content: currentBubble }];
          }
          return [...prev, { role: "assistant", content: currentBubble }];
        });
      }
    }
  }
}
```

## 2️⃣ `event: card` — 结构化数据卡片

当 Agent 调用工具获取到结构化数据（课表、成绩、记录列表等）时，以卡片形式独立返回，**与文本消息分离**。

### 格式

```
event: card
data: {
  "title": "📅 今日课表",
  "type": "schedule",          // 卡片类型标识
  "rows": [
    { "label": "📚 课程", "value": "高等数学" },
    { "label": "⏰ 时间", "value": "08:30-10:00" },
    { "label": "📍 地点", "value": "教学楼301" },
    { "label": "👨‍🏫 教师", "value": "王老师" }
  ]
}
```

### 卡片类型定义

| type | 含义 | rows 示例 |
|:-----|:-----|:---------|
| `schedule` | 课表 | 课程/时间/地点/教师 |
| `checkin_result` | 签到结果 | 课程/状态/时间 |
| `checkin_record` | 签到记录列表 | 每条记录一行 |
| `homework` | 作业信息 | 标题/截止/状态 |
| `homework_list` | 作业列表 | 多条作业 |
| `grade` | 成绩 | 分数/评语 |
| `leave` | 请假 | 课程/原因/状态 |
| `info` | 个人信息 | 姓名/学号/班级/手机 |
| `notif` | 通知 | 标题/内容/时间 |
| `list` | 通用列表 | 多条项目 |
| `confirm` | 确认操作 | 操作描述/确认按钮 |

### 完整示例 — 课表卡片

```
event: message
data: {"content": "查到了～这是你今天的课程安排："}

event: card
data: {
  "title": "📅 周三 6月26日",
  "type": "schedule",
  "rows": [
    { "label": "08:30-10:00", "value": "高等数学 @ 教学楼301" },
    { "label": "10:20-11:50", "value": "大学英语 @ 教学楼205" },
    { "label": "14:00-15:30", "value": "数据结构 @ 实验楼403" }
  ]
}

event: done
data: {}
```

### 前端渲染

```tsx
// 卡片组件
function CardMessage({ card }: { card: CardData }) {
  const iconMap: Record<string, string> = {
    schedule: "📅",
    checkin_result: "📍",
    homework: "📝",
    grade: "📊",
    leave: "📋",
    info: "👤",
    notif: "🔔",
  };

  return (
    <div className="msg-card">
      <div className="card-header">
        <span className="card-icon">{iconMap[card.type] || "📋"}</span>
        <span className="card-title">{card.title}</span>
      </div>
      <div className="card-body">
        {card.rows.map((row, i) => (
          <div key={i} className="card-row">
            <span className="card-label">{row.label}</span>
            <span className="card-value">{row.value}</span>
          </div>
        ))}
      </div>
    </div>
  );
}
```

## 3️⃣ `event: tool_call` — 工具调用状态

当 Agent 调用 MCP 工具时，通知前端显示"正在处理"状态。

### 格式

```
event: tool_call
data: {
  "name": "do_checkin",
  "args": { "student_id": "20260001" },
  "status": "executing"    // executing | success | error
}
```

### 完整示例

```
// Agent 正在调用工具
event: tool_call
data: {"name": "do_checkin", "args": {"student_id": "20260001"}, "status": "executing"}

// 工具执行完成，LLM 正在生成回复（静默）
event: tool_call
data: {"name": "do_checkin", "status": "success"}

// LLM 生成回复
event: message
data: {"content": "✅ 签到成功！高等数学 @ 教学楼301"}

event: done
data: {}
```

### 前端处理

```typescript
// 显示"正在处理"指示器
function ChatMessages({ messages }) {
  return (
    <div className="chat-messages">
      {messages.map((msg, i) => (
        <div key={i} className={`bubble ${msg.role}`}>
          {msg.type === "card" ? (
            <CardMessage card={msg.card} />
          ) : (
            msg.content
          )}
        </div>
      ))}
      {isToolExecuting && (
        <div className="bubble assistant typing">
          🤖 正在处理...
        </div>
      )}
    </div>
  );
}
```

## 4️⃣ `event: error` — 错误响应

### 格式

```
event: error
data: {
  "code": "NO_COURSE",
  "message": "当前没有进行中的课程，无法签到"
}
```

### 错误码定义

| code | 含义 | 说明 |
|:-----|:-----|:------|
| `NO_COURSE` | 无进行中课程 | 签到/课表查询时当前时间无课 |
| `ALREADY_CHECKIN` | 已签到过 | 今天已签到该课程 |
| `NO_STUDENT` | 未找到学生 | 学号不存在 |
| `SERVER_ERROR` | 服务端错误 | 后端/MCP 异常 |
| `AUTH_EXPIRED` | 登录过期 | Token 失效，需重新登录 |
| `PARAM_MISSING` | 参数不足 | Agent 需要补充参数 |
| `MCP_ERROR` | MCP 调用失败 | 校园端 MCP Server 异常 |

### 前端处理

```typescript
// 显示错误 toast，不推气泡
function handleError(code: string, message: string) {
  if (code === "AUTH_EXPIRED") {
    // 跳转登录页
    localStorage.removeItem("token");
    window.location.href = "/login";
    return;
  }
  // Toast 显示错误
  showToast(message, "error");
}
```

### 格式

```

```

```typescript
// 停止流式接收，恢复输入框
```

## 7️⃣ 消息历史格式（用于多轮上下文）

前端维护的消息历史，每次请求发送给后端：

```typescript
interface Message {
  role: "user" | "assistant" | "tool";
  content?: string;        // 文本内容（user/assistant）
  card?: CardData;         // 卡片数据（assistant）
  tool_calls?: ToolCall[]; // 工具调用（assistant，可选）
  name?: string;           // 工具名（tool）
}

interface CardData {
  title: string;
  type: string;
  rows: { label: string; value: string }[];
}

interface ToolCall {
  id: string;
  type: "function";
  function: {
    name: string;
    arguments: string;  // JSON string
  };
}
```

### 示例——发送的历史

```json
{
  "messages": [
    { "role": "user", "content": "帮我签到" },
    {
      "role": "assistant",
      "content": null,
      "tool_calls": [{
        "id": "call_xxx",
        "type": "function",
        "function": { "name": "do_checkin", "arguments": "{\"student_id\":\"20260001\"}" }
      }]
    },
    { "role": "tool", "name": "do_checkin", "content": "✅ 签到成功！高等数学" },
    { "role": "assistant", "content": "✅ 签到成功！高等数学 @ 教学楼301" },
    { "role": "user", "content": "看看我的成绩" }
  ],
  "student_id": "20260001"
}
```

## 8️⃣ 统一返回结构（非 SSE 接口）

非流式接口（登录、注册、操作记录等）统一返回格式：

```json
{
  "status": "success",
  "data": {},
  "msg": "操作成功"
}
```

| 字段 | 类型 | 说明 |
|:-----|:-----|:------|
| `status` | string | `success` / `error` |
| `data` | object | 业务数据 |
| `msg` | string | 提示信息 |

## 🔧 前端 SSE 接收完整示例

```typescript
// api/chat.ts
async function sendChatMessage(
  messages: Message[],
  studentId: string,
  callbacks: {
    onMessage?: (content: string) => void;
    onCard?: (card: CardData) => void;
    onToolCall?: (tool: ToolCallInfo) => void;
    onError?: (code: string, msg: string) => void;
    onDone?: () => void;
  }
) {
  const resp = await fetch("/api/chat", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ messages, student_id: studentId }),
  });

  const reader = resp.body!.getReader();
  const decoder = new TextDecoder();
  let buffer = "";

  while (true) {
    const { done, value } = await reader.read();
    if (done) break;

    buffer += decoder.decode(value, { stream: true });
    const lines = buffer.split("\n");
    buffer = lines.pop() || "";  // 保留未完成的行

    for (const line of lines) {
      if (line.startsWith("event: ")) {
        currentEvent = line.slice(7).trim();
      } else if (line.startsWith("data: ")) {
        const data = JSON.parse(line.slice(6));
        switch (currentEvent) {
          case "message":
            callbacks.onMessage?.(data.content);
            break;
          case "card":
            callbacks.onCard?.(data);
            break;
          case "tool_call":
            callbacks.onToolCall?.(data);
            break;
          case "error":
            callbacks.onError?.(data.code, data.message);
            break;
          case "done":
            callbacks.onDone?.();
            break;
        }
      }
    }
  }
}
```
