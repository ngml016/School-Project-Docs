# 🎒 学生端 H5 开发文档

> 学生端是一个 **H5 移动端页面（兼容 PC）**，以 **对话 Agent 为主界面**，功能页面辅助操作。
> 所有功能（Agent 对话 + 页面操作）共用校园端 MCP 接口。

---

## 🏗️ 架构总览

```
学生手机浏览器 / PC 浏览器
      │
      ├── 聊天页（主页）：Agent 对话完成所有功能
      │   └── 快捷入口 → 跳转功能页面
      │
      └── 功能页面：特定操作（签到/课表/作业/设置...）
          └── 调用 → 校园端 MCP

所有操作 → MCP 协议 → 校园端 MCP Server (:8001)
```

**核心原则：**
- **聊天页 = 主页**，默认打开，所有功能都能通过对话完成
- **功能页面 = 辅助**，从聊天页快捷入口进入，提供更好的操作体验
- **接口统一**，对话和页面调的是同一套 MCP 工具

## 📱 H5 优先 + PC 兼容设计

### 断点策略

| 断点 | 宽度 | 布局模式 | 说明 |
|:----|:----|:---------|:-----|
| 手机 | < 768px | **单栏全屏** | 聊天占满整个屏幕，底栏固定 |
| 平板 | 768px ~ 1024px | 居中窄栏（max-width: 480px） | 聊天面板居中，背景显示 |

### MCP 调用封装

```typescript
// api/mcp.ts
const MCP_BASE = '/api/mcp';  // 由 dev proxy 转发到校园端 :8001

export interface MCPResult {
  success: boolean;
  data?: any;
  message: string;
}

export async function callTool(tool: string, args: Record<string, any>): Promise<MCPResult> {
  const resp = await fetch(`${MCP_BASE}/tools/call`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ tool, arguments: args }),
  });
  return resp.json();
}
```

### Agent 调用流程

```typescript
// agent/orchestrator.ts
async function processMessage(input: string, files?: File[]) {
  let fileUrl = '';

  // 1. 如果有文件，先上传
  if (files?.length) {
    const base64 = await fileToBase64(files[0]);
    const result = await callTool('upload_file', {
      file_data: base64,
      filename: files[0].name,
      upload_type: 'homework',
    });
    fileUrl = result.data?.url || '';
  }

  // 2. 调 LLM（带 function calling）
  const response = await callLLM([
    { role: 'system', content: SYSTEM_PROMPT },
    ...messageHistory,
    { role: 'user', content: fileUrl ? `${input}\n[已上传文件：${fileUrl}]` : input },
  ]);

  // 3. 处理工具调用或直接回复
  if (response.tool_call) {
    const toolResult = await callTool(response.tool_call.name, response.tool_call.args);
    const final = await callLLM([
      ...messages,
      { role: 'assistant', content: null, tool_calls: [response.tool_call] },
      { role: 'tool', content: JSON.stringify(toolResult), tool_call_id: response.tool_call.id },
    ]);
    return final.content;
  }

  return response.content;
}
```

## 🌐 跨域处理

学生端 H5 开发时需要跨域调校园端 MCP Server。

### Vite dev proxy（推荐）

```typescript
// vite.config.ts
export default defineConfig({
  server: {
    proxy: {
      '/api/mcp': {
        target: 'http://localhost:8001',
        changeOrigin: true,
        rewrite: path => path.replace(/^\/api\/mcp/, ''),
      },
      '/uploads': {
        target: 'http://localhost:8000',
        changeOrigin: true,
      },
    },
  },
});
```
