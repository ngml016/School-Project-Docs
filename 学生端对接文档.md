# 🎒 学生端开发文档

> 学生端是一个独立的客户端项目（App / 小程序 / Web），通过 **MCP 协议** 对接校园端，完成签到、作业、课表查询等校园事务。

---

## 📋 学生端需要实现的功能

### 1. 用户体系（学生自己完成）

| 功能 | 说明 | 对接 MCP |
|:----|:-----|:--------:|
| 注册 | 输入手机号、密码、姓名，选择班级 | ✅ |
| 登录 | 输入学号/手机号 + 密码，获取 token | ✅ |
| 个人中心 | 展示姓名、学号、班级、手机号 | ✅ |

### 2. 课表与课程

| 功能 | 说明 | 对接 MCP |
|:----|:-----|:--------:|
| 今日课表 | 展示当天课程（时间/名称/地点/教师） | ✅ |
| 指定日课表 | 查看周一到周日的课程安排 | ✅ |
| 已选课程 | 查看本学期所有课程列表 | ✅ |

### 3. 签到

| 功能 | 说明 | 对接 MCP |
|:----|:-----|:--------:|
| 签到 | 输入教师发布的6位签到码 | ✅ |
| 签到记录 | 查看最近N天的签到历史 | ✅ |
| 出勤统计 | 看出勤率/迟到/缺勤统计 | ✅ |

### 4. 作业

| 功能 | 说明 | 对接 MCP |
|:----|:-----|:--------:|
| 待交作业 | 查看未提交、未截止的作业 | ✅ |
| 全部作业 | 查看所有作业（待交/已交/已批改分类） | ✅ |
| 作业详情 | 查看作业描述、截止时间、提交状态 | ✅ |
| 提交作业 | 输入文字内容提交作业 | ✅ |
| 作业成绩 | 查看已批改的作业分数和评语 | ✅ |

### 5. 请假

| 功能 | 说明 | 对接 MCP |
|:----|:-----|:--------:|
| 请假申请 | 选择课程、填写原因，提交请假 | ✅ |

### 6. 通知与公告

| 功能 | 说明 | 对接 MCP |
|:----|:-----|:--------:|
| 通知列表 | 查看系统推送的个人通知 | ✅ |
| 班级公告 | 查看教师发布的班级/全校公告 | ✅ |

## 🛠️ 全部 MCP 工具（17 个）

### 注册 & 登录

| # | 工具 | 参数 | 返回 |
|---|------|------|------|
| 1 | `get_class_list()` | 无 | 班级列表（ID + 名称） |
| 2 | `student_register(phone, password, name, class_id)` | 手机号、密码、姓名、班级ID | ✅ 注册成功 + 学号 |
| 3 | `student_login(account, password)` | 手机号或学号、密码 | ✅ 登录成功 + Token |

### 信息 & 课表

| # | 工具 | 参数 | 返回 |
|---|------|------|------|
| 4 | `get_student_info(student_id)` | 学号 | 姓名、班级、手机号 |
| 5 | `get_schedule(student_id, day_of_week?)` | 学号、星期（0=今天） | 课表列表 |
| 6 | `get_courses(student_id)` | 学号 | 已选课程列表 |

### 签到

| # | 工具 | 参数 | 返回 |
|---|------|------|------|
| 7 | `do_checkin(student_id, code)` | 学号、6位签到码 | ✅ 签到成功/失败 |
| 8 | `get_checkin_records(student_id, days?)` | 学号、天数(默认7) | 签到记录列表 |
| 9 | `get_checkin_stats(student_id, days?)` | 学号、天数(默认30) | 出勤统计 |

### 作业

| # | 工具 | 参数 | 返回 |
|---|------|------|------|
| 10 | `get_pending_homeworks(student_id)` | 学号 | 待交作业列表 |
| 11 | `get_all_homeworks(student_id)` | 学号 | 全部作业（分类） |
| 12 | `get_homework_detail(homework_id, student_id?)` | 作业ID、学号(可选) | 作业详情 |
| 13 | `submit_homework(student_id, homework_id, content)` | 学号、作业ID、内容 | ✅ 提交成功/失败 |
| 14 | `get_homework_grades(student_id)` | 学号 | 作业成绩列表 |

### 请假

| # | 工具 | 参数 | 返回 |
|---|------|------|------|
| 15 | `apply_leave(student_id, course_id, reason)` | 学号、课程ID、原因 | ✅ 提交成功 |

### 通知

| # | 工具 | 参数 | 返回 |
|---|------|------|------|
| 16 | `get_notifications(student_id)` | 学号 | 通知列表 |
| 17 | `get_announcements(class_id)` | 班级ID | 公告列表 |

> 参数后带 `?` 表示可选参数

## 🔌 MCP 对接方式

### 方式一：SSE 模式（推荐，通过网络连接）

学生端配置 MCP 客户端连接到校园端服务器：

```
MCP Server URL: http://<校园服务器IP>:8001/sse
```

**示例配置（Claude Desktop / Cursor / VS Code）：**

```json
{
  "mcpServers": {
    "campus-agent": {
      "type": "sse",
      "url": "http://192.168.1.100:8001/sse"
    }
  }
}
```

### 方式二：自定义客户端（直接调 HTTP）

MCP SSE 协议步骤：

```
1. GET  http://server:8001/sse
   → 返回 session_id: /messages/?session_id=xxx

2. POST http://server:8001/messages/?session_id=xxx
   Content-Type: application/json
   Body: JSON-RPC 格式的工具调用请求

3. 通过 SSE 连接接收工具返回结果
```

### 方式三：stdio 模式（本地子进程）

```bash
python mcp_server.py --transport stdio
```

学生端 Agent 以子进程方式启动 MCP Server，通过 stdin/stdout 通信。

## 🎯 对接示例流程

### 注册流程

```
学生端 → get_class_list()
       ← 返回班级列表

学生选班级 → student_register("138xxxx", "123456", "张三", 1)
           ← 返回 ✅ 注册成功！学号：20260001
```

### 签到流程

```
教师端发起签到 → 学生看到签到码

学生输入码 → do_checkin("20260001", "731659")
           ← 返回 ✅ 签到成功！《高等数学》
```

### 作业流程

```
学生打开作业页 → get_pending_homeworks("20260001")
               ← 待交作业列表

学生查看详情 → get_homework_detail(1, "20260001")
             ← 作业详情（描述/截止时间/提交状态）

学生提交   → submit_homework("20260001", 1, "这是作业答案")
           ← ✅ 作业已提交
```

## ⚙️ 学生端架构建议

```
┌─────────────────────────────────┐
│         学生端 App/小程序         │
│  ┌───────────────────────────┐  │
│  │    学生 LLM Agent         │  │ ← 识别意图、调用工具
│  └──────────┬────────────────┘  │
│             │ MCP 协议           │
│  ┌──────────▼────────────────┐  │
│  │    MCP Client             │  │ ← 连接校园端 MCP Server
│  └──────────┬────────────────┘  │
└─────────────┼───────────────────┘
              │ HTTP (SSE)
┌─────────────▼───────────────────┐
│    校园端 MCP Server (:8001)     │ ← 17 个工具
│    校园端 FastAPI (:8000)        │ ← 管理后台
│    SQLite 数据库                 │
└─────────────────────────────────┘
```

学生端 Agent 的职责：
1. **理解学生意图**（"帮我签到"）
2. **决定调哪个 MCP 工具**（do_checkin）
3. **收集参数**（问签到码）
4. **调用工具并展示结果**

## ⚠️ 注意事项

| 注意点 | 说明 |
|:------|:-----|
| 🔑 **学号自动生成** | 注册时不用填学号，系统返回 |
| 📍 **签到码6位数字** | 由教师端发起签到生成，有效期默认10分钟 |
| 📝 **作业提交文字** | 目前支持文字内容，文件上传需后续扩展 |
| 🪪 **Token 有效期** | 登录返回的 token 7天有效 |
| 🌐 **两个服务** | 校园端运行两个端口：8000（管理后台）、8001（MCP） |
| 🔄 **实时推送** | 学生签到/交作业/请假后，校园端右上角会收到 toast 推送 |
